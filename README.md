# News-App

App đọc tin tức đơn giản, tin cậy, ổn định cho Android. <br>
Nguồn tin: https://vnexpress.net

## Features
* Lọc tin tức theo các danh mục
* Tin tức được cập nhật tới thời điểm truy cập
* Nguồn tin tin cậy
* Tôc độ ổn định

## Thư viện
* XMLDomParser
* Recycler Adapter
* Picasso API
* ...
